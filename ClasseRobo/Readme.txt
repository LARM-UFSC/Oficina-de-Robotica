Copie a pasta "ClasseRobo" para a pasta "Documentos\Arduino". 
Em seguida abra o IDE do Arduino. Se quiser acessar os exemplos
vá no menu "file -> Examples -> ClasseRobo". Existem três exemplos,
um para testar o sensor de ultrassom, o outro para testar os motores e o último
um exemplo de controle básico.
